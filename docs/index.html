ヒストグラム：棒の長さで連続変数の分布を表現する

ヒストグラムは「棒の間の隙間を空けない」のが通例

棒グラフ: 項目が独立しているので、隙間を空ける。

ヒストグラム: 数値が連続していることを示すために、棒をくっつける。

ggplot(diamonds) +
  geom_histogram(aes(x = carat), bins = 20)
